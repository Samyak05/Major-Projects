## 🌐 01. TLS Handshake Analysis over Layer-3 Routing (Linux Network Namespaces)

Experimental networking project built to deeply understand how TCP and TLS operate across routed network segments.  
This project simulates isolated hosts and routers using Linux network namespaces and analyzes TLS handshakes across Layer-3 boundaries.

🔗 Repository:  
https://github.com/Samyak05/TLS-Handshake-Linux-Namespaces.git

Focus Areas:
- Linux Network Namespaces
- IP routing & packet forwarding
- TCP 3-way handshake observation
- TLS handshake packet analysis
- Wireshark & tcpdump tracing
- Layer-3 network isolation
